# AI Career Advisor Chatbot
A Prompt Engineering project that helps final-year students choose an AI career path and generates a personalized 6-month roadmap
## Problem
Many final-year students are confused about which AI career path to choose.
## Solution
This project uses Prompt Engineering to:
- Analyze student interests and goals
- Recommend a suitable AI career path
- Generate a personalized 6-month roadmap
## Skills Used
- Prompt Engineering
- Prompt Chaining
- Multi-Step Prompting
- Structured Output Prompting
- Generative AI
## Target Users
- Final-year students
- AI beginners
- Career changers
